<span data-ttu-id="845a5-118">Los componentes del sistema Microsoft 365 se alojan en centros de los centros de recursos separados geográficamente de los equipos de operaciones.</span><span class="sxs-lookup"><span data-stu-id="845a5-118">Microsoft 365 system components are housed in datacenters geographically separated from the operations teams.</span></span> <span data-ttu-id="845a5-119">El personal del centro de recursos no tiene acceso lógico a los sistemas Microsoft 365.</span><span class="sxs-lookup"><span data-stu-id="845a5-119">Datacenter personnel do not have logical access to Microsoft 365 systems.</span></span> <span data-ttu-id="845a5-120">Como resultado, el personal del equipo de servicio de 365 de Microsoft administra el entorno a través de acceso remoto.</span><span class="sxs-lookup"><span data-stu-id="845a5-120">As a result, Microsoft 365 service team personnel manage the environment through remote access.</span></span> <span data-ttu-id="845a5-121">El personal del equipo de servicio que requiere acceso remoto a soporte técnico de Microsoft 365 solo se le concede acceso remoto tras la aprobación de un administrador autorizado.</span><span class="sxs-lookup"><span data-stu-id="845a5-121">Service team personnel who require remote access to support Microsoft 365 are only granted remote access after approval from an authorized manager.</span></span> <span data-ttu-id="845a5-122">Todos los accesos remotos usan TLS compatible con FIPS 140-2 para conexiones remotas seguras.</span><span class="sxs-lookup"><span data-stu-id="845a5-122">All remote access uses FIPS 140-2 compatible TLS for secure remote connections.</span></span>

<span data-ttu-id="845a5-123">Microsoft 365 usa estaciones de trabajo de administración seguras (Sierras) para el acceso remoto del equipo de servicio para ayudar a proteger los entornos de Microsoft 365 de la intromisión.</span><span class="sxs-lookup"><span data-stu-id="845a5-123">Microsoft 365 uses Secure Admin Workstations (SAWs) for service team remote access to help protect Microsoft 365 environments from compromise.</span></span> <span data-ttu-id="845a5-124">Estas estaciones de trabajo están diseñadas específicamente para evitar la pérdida intencionada o no de datos de producción, como el bloqueo de puertos USB y la limitación del software disponible en la sierra a lo que se necesita para admitir el entorno.</span><span class="sxs-lookup"><span data-stu-id="845a5-124">These workstations are specifically designed to prevent intentional or unintentional loss of production data, including locking down USB ports and limiting the software available on the SAW to what is required for supporting the environment.</span></span> <span data-ttu-id="845a5-125">Se realiza un seguimiento y supervisión de los Sierras para detectar y evitar un riesgo malintencionado o accidental de los datos de los clientes por parte de los ingenieros de Microsoft.</span><span class="sxs-lookup"><span data-stu-id="845a5-125">SAWs are closely tracked and monitored to detect and prevent malicious or inadvertent compromise of customer data by Microsoft engineers.</span></span>
