<span data-ttu-id="33696-p102">El Reglamento general de protección de datos (RGPD) exige a los responsables de los datos realizar una evaluación del impacto de la protección de datos (EIPD) en el caso de operaciones de procesamiento que puedan "suponer un alto riesgo para los derechos y libertades de las personas físicas". No hay nada inherente en el servicio de procesamiento de datos para Windows Enterprise que necesariamente exija al responsable de los datos la creación de una EIPD al usar dicho servicio. Rather, whether a DPIA is required will be dependent on the details and context of how the data controller deploys, configures, and uses the data processor service for Windows Enterprise.</span></span> 

<span data-ttu-id="33696-110">La finalidad de este documento es proporcionar a los responsables de los datos información sobre el servicio de procesamiento de datos para Windows Enterprise que les permitirá determinar si es necesario realizar una EIPD y, en caso afirmativo, qué detalles incluir en la misma.</span><span class="sxs-lookup"><span data-stu-id="33696-110">The purpose of this document is to provide data controllers with information about the data processor service for Windows Enterprise that will help them to determine whether a DPIA is needed and, if so, what details to include.</span></span>
