<span data-ttu-id="de1ce-p102">**Servicio de itinerancia de Office**: la itinerancia de Office es un servicio que almacena la configuración relacionada con Office (como el tema de Office, el diccionario personalizado, la configuración del idioma, el modo de programador y la autocorrección). <span data-ttu-id="de1ce-p103">**Workplace Analytics**: la herramienta de exportación de datos de registro proporciona datos de uso relativos a aquellos usuarios de la organización que tienen permiso para ejecutar informes de Workplace Analytics. Workplace Analytics también calcula y almacena los datos seudonimizados derivados de datos de Office 365 para mejorar el rendimiento. Si quiere que estos datos seudonimizados estén disponibles para un usuario y necesita ayuda para ello, póngase en contacto con el equipo de [soporte técnico de Microsoft](https://support.microsoft.com/contactus/).</span><span class="sxs-lookup"><span data-stu-id="de1ce-p103">**Workplace Analytics**: The data log export tool provides usage data for those users in your organization who have permission to run Workplace Analytics reports. Workplace Analytics also computes and stores pseudonymized data derived from Office 365 data to improve performance. If you would like to make this pseudonymized data available to a user and need assistance, contact [Microsoft Support](https://support.microsoft.com/contactus/).</span></span>

<span data-ttu-id="de1ce-p104">**Yammer**: el Centro de administración de Yammer permite exportar los datos de actividad de una cuenta de usuario. For detailed instructions, see [Manage GDPR data subject requests in Yammer Enterprise](/yammer/manage-security-and-compliance/gdpr-requests-in-yammer-enterprise).</span></span>
