# <a name="azure-and-dynamics-365-accountability-readiness-checklist-for-the-gdpr"></a>Lista de comprobación de preparación de responsabilidad de Azure y Dynamics 365 para el RGPD

## <a name="1-introduction"></a>1. Introducción

Esta lista de comprobación de preparación de responsabilidad ofrece una forma cómoda de acceso a la información que puede que sea necesaria para cumplir con el Reglamento general de protección de datos (RGPD) al usar Microsoft Azure y Dynamics 365. Esta lista se organiza con los títulos y números de referencia (entre paréntesis para cada tema de la lista de comprobación) de un conjunto de controles de seguridad y privacidad de los encargados del tratamiento de datos personales, extraídos de:

- [ISO/CEI 27701](https://shop.bsigroup.com/ProductDetail?pid=000000000030351736) para los requisitos de administración de la privacidad.
- [ISO/CEI 27001](https://shop.bsigroup.com/ProductDetail?pid=000000000030347472) para los requisitos de técnicas de seguridad.

Los elementos de esta lista de comprobación se pueden administrar con el Administrador de cumplimiento [[16](gdpr-arc-azure-dynamics.md#16)]; para ello, hay que hacer referencia al identificador de control y al título de control que figuran en la sección de controles administrados por el cliente del RGPD. Esta misma estructura de controles se usa también para organizar la presentación de los controles internos que Microsoft Azure y Dynamics 365 implementan para cumplir con el RGPD, que se puede descargar en el [Portal de confianza de servicios](https://servicetrust.microsoft.com/ViewPage/TrustDocuments).

Esta estructura de control también se usa para organizar la presentación de los [controles internos que Microsoft Dynamics 365 implementa para cumplir con el RGPD](https://aka.ms/gdprcontrols).

Para obtener más documentación relacionada con el RGPD, visite [https://aka.ms/gdprgetstarted](https://aka.ms/gdprgetstarted).
